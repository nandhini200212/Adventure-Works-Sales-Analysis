Adventure Works Sales Analysis

This project analyzes the Adventure Works dataset to uncover insights into sales performance, customer behavior, and product trends. Using Power BI, Tableau, SQL, and Excel, interactive dashboards and reports were created to monitor key business metrics and support data-driven decision-making.

Tools Used:

Power BI
Tableau
SQL
Microsoft Excel

Key Features:

Sales Performance Analysis
Customer Analysis
Product Category Analysis
Profit and Revenue Tracking
KPI Dashboard Development
Data Cleaning and Transformation
Interactive Visualizations and Reporting

Key KPIs:

Total Sales
Total Profit
Total Orders
Customer Performance
Product Performance
Regional Sales Analysis

Outcome:
Generated actionable insights that helped identify business trends, evaluate sales performance, and support strategic decision-making.
